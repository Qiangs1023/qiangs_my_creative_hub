我把 2024 年所有发票、所有支出、所有税款列了一张表。
最让我意外的不是数字本身，而是结构。

## 收入构成（净收入约 ¥780k）

| 来源 | 占比 |
| ---- | ---- |
| 课程销售 | 38% |
| SaaS 订阅 | 32% |
| 赞助与合作 | 18% |
| 出售项目（一次性） | 12% |

订阅收入"看起来"应该是大头，
但实际上课程在每次更新的时候会带来一波集中的现金流。

## 支出 ~¥120k

最大的三块：
- 软件订阅：~¥18k（Figma / Notion / Linear / OpenAI / Cursor / 服务器）
- 外包：~¥40k（剪辑、字幕、客服）
- 税：~¥58k

## 我交的学费

- 2023 年报税方式选错，多交了 ~¥15k
- 给一个项目做了 6 个月，最后零收入
- 一次过早签的合作协议，让我后来推掉了一个更大的机会

## 结论

收入越分散，睡得越好。
透明地写下来这件事本身，比赚钱更让我安心。
